================================================================================
MUSICAR2 v1.0 - NUEVA VERSIÓN CON SELECCIÓN MANUAL
================================================================================

🎯 OBJETIVO PRINCIPAL:
Permitir seleccionar canciones UNA POR UNA de tu biblioteca,
editarlas con nombres personalizados,
mantenerlas en un orden exacto,
y enviarlas a una carpeta "Ordenada" para reproducirse en el orden deseado
en tu USB o tarjeta SD del coche.

================================================================================
NUEVA INTERFAZ - 3 PANELES:
================================================================================

PANEL IZQUIERDO - BIBLIOTECA DE MÚSICA
─────────────────────────────────────────
• Botón "Examinar Carpeta"
  → Abre un diálogo para elegir tu carpeta de música
  → Carga automáticamente todos los archivos MP3, WMA, WAV, FLAC, M4A
  → Muestra lista de canciones disponibles

PANEL CENTRAL - BOTONES DE CONTROL
──────────────────────────────────
• >> Agregar
  → Selecciona una canción de la biblioteca izquierda
  → La añade a la lista de seleccionadas (panel derecho)
  → Asigna número secuencial (001, 002, 003...)

PANEL DERECHO - CANCIONES SELECCIONADAS
──────────────────────────────────────
• Muestra todas las canciones en orden de reproducción
• Formato: "001 - Nombre Canción.mp3"
• Selecciona una canción para editar su nombre

BOTONES DE ORDEN:
• ↑ Arriba   → Mueve canción arriba en el orden
• ↓ Abajo    → Mueve canción abajo en el orden
• Remover    → Elimina la canción de la lista

EDITOR DE NOMBRE:
• Campo de texto para editar el nombre
• Botón "Actualizar Nombre" → Guarda el cambio
• Se carga automáticamente al seleccionar una canción

BOTÓN PRINCIPAL:
• ✓ Enviar a Carpeta "Ordenada"
  → Copia todos los archivos a: C:\Users\[TuUsuario]\Music\Ordenada
  → Los nombra con números secuenciales: 001 - nombre.mp3, 002 - nombre.mp3...
  → Preserva las extensiones originales
  → Crea la carpeta automáticamente si no existe

================================================================================
FLUJO DE TRABAJO - PASO A PASO:
================================================================================

PASO 1: CARGAR BIBLIOTECA
├─ Inicia la aplicación: java -cp "build\classes;src" musicar2.FramePrincipal
├─ Haz clic en "Examinar Carpeta"
├─ Navega a tu carpeta de música
└─ Observa la lista de canciones disponibles en el panel izquierdo

PASO 2: AGREGAR CANCIONES (UNA POR UNA)
├─ Selecciona una canción en el panel izquierdo
├─ Haz clic en ">> Agregar"
├─ La canción aparece en la lista derecha con número (001 - ...)
├─ Repite para cada canción que desees
└─ El número se asigna automáticamente en orden

PASO 3: ORGANIZAR EL ORDEN (OPCIONAL)
├─ Si deseas cambiar el orden:
├─ Selecciona una canción en el panel derecho
├─ Usa "↑ Arriba" o "↓ Abajo" para reposicionarla
├─ Los números se actualizan automáticamente
└─ Los botones también se renumeran correctamente

PASO 4: EDITAR NOMBRES (OPCIONAL)
├─ Selecciona una canción en el panel derecho
├─ El nombre aparece en el campo de texto
├─ Modifica el nombre como desees (ej: "Mi Canción Favorita.mp3")
├─ Haz clic en "Actualizar Nombre"
└─ El cambio se refleja inmediatamente

PASO 5: REMOVER CANCIONES (SI NECESITAS)
├─ Selecciona una canción en el panel derecho
├─ Haz clic en el botón rojo "Remover"
├─ Se elimina de la lista
├─ Los números se reordenaran automáticamente
└─ El archivo original NO se borra, solo se quita de la selección

PASO 6: ENVIAR A CARPETA "ORDENADA"
├─ Cuando estés satisfecho con el orden y los nombres:
├─ Haz clic en el gran botón verde: "✓ Enviar a Carpeta 'Ordenada'"
├─ Los archivos se copian a: C:\Users\[TuUsuario]\Music\Ordenada
├─ Se nombran así: "001 - nombre.mp3", "002 - nombre.mp3", etc.
├─ Se muestra un resumen con los archivos copiados
└─ ¡Listo para usar en tu USB o SD del coche!

PASO 7: USAR EN EL COCHE
└─ Copia la carpeta "Ordenada" a tu USB o tarjeta SD
   Las canciones se reproducirán en el orden EXACTO que estableciste

CARACTERÍSTICAS PRINCIPALES:
================================================================================

✓ Selección manual de canciones una por una
✓ Edición de nombres directamente en la interfaz
✓ Reordenamiento visual con botones arriba/abajo
✓ Números secuenciales automáticos (001, 002, 003...)
✓ Lista visual clara del orden de reproducción
✓ Interfaz intuitiva y fácil de usar
✓ Soporte para múltiples formatos: MP3, WMA, WAV, FLAC, M4A
✓ Carpeta "Ordenada" creada automáticamente
✓ Copia segura (no mueve, copia los archivos)

================================================================================
FORMATOS SOPORTADOS:
================================================================================

✓ MP3   (.mp3)
✓ WMA   (.wma)
✓ WAV   (.wav)
✓ FLAC  (.flac)
✓ M4A   (.m4a)

Se pueden agregar fácilmente más formatos editando la línea:
    "Archivos de Música", "mp3", "wma", "wav", "flac", "m4a"

================================================================================
UBICACIÓN DE SALIDA:
================================================================================

Los archivos se copian automáticamente a:
C:\Users\[TuNombreUsuario]\Music\Ordenada

Ejemplos:
• C:\Users\Juan\Music\Ordenada
• C:\Users\Maria\Music\Ordenada
• C:\Users\PC-NOX\Music\Ordenada

Si no existe la carpeta "Ordenada", se crea automáticamente.
Si existe, copia los archivos sin preguntar.

================================================================================
NOTAS IMPORTANTES:
================================================================================

⚠ Los archivos se COPIAN, no se MUEVEN
  → Los originales permanecen en su ubicación
  → Se crea una copia en la carpeta "Ordenada"

⚠ Los nombres se pueden editar libremente
  → Solo edita el nombre, NO agregues barras (\) o otros caracteres especiales
  → Mantén las extensiones .mp3, .wma, etc.

⚠ El orden importa
  → Los números determinan el orden de reproducción
  → Los coches reproducen en orden numérico (001, 002, 003...)
  → Asegúrate de que el orden sea exacto antes de enviar a "Ordenada"

⚠ La carpeta "Ordenada" se puede limpiar
  → Si vuelves a usar la aplicación, sobrescribe los archivos
  → Haz un backup si tienes contenido importante

================================================================================
RESOLUCIÓN DE PROBLEMAS:
================================================================================

P: No veo mis archivos MP3
R: Verifica que:
   1. Estén en formato .mp3, .wma, .wav, .flac o .m4a
   2. Estén en la carpeta que seleccionaste
   3. No estén protegidos o en una carpeta del sistema

P: La carpeta "Ordenada" no se crea
R: Verifica que:
   1. Tengas permisos de escritura en C:\Users\[Tu usuario]\Music
   2. No haya una carpeta "Ordenada" existente protegida
   3. El disco tenga espacio disponible

P: Los archivos no se copian
R: Verifica que:
   1. Los archivos originales no estén abiertos en otro programa
   2. Tengas espacio en disco
   3. Los archivos no estén protegidos

P: El coche no reproduce en orden correcto
R: Recuerda que:
   1. El nombre debe ser: 001.mp3, 002.mp3, 003.mp3 (solo números)
   2. O: 001 - nombre.mp3, 002 - nombre.mp3 (MusiCar2 usa este formato)
   3. Algunos coches viejos solo reconocen números sin guiones
